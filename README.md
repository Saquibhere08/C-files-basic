## Getting Started

To run any program in this repository:

1. Clone the repo:  
   ```bash
   git clone https://github.com/yourusername/c-programs.git
   ```
2. Navigate to the program directory.
3. Compile with GCC:  
   ```bash
   gcc program_name.c -o output
   ```
4. Run the program:  
   ```bash
   ./output
   ```
